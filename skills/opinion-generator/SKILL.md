# 观点生成器

基于调研素材，从以下 10 个角度各生成一个观点。

## 10 个角度

| # | 角度 | 思考方式 |
|---|------|---------|
| 1 | 反常识 | 大众怎么看？反过来呢？ |
| 2 | 二阶效应 | 直接后果大家都知道，间接后果呢？ |
| 3 | 利益分析 | 谁受益？谁受损？为什么？ |
| 4 | 历史类比 | 这件事历史上发生过吗？结果如何？ |
| 5 | 技术本质 | 抛开营销，技术上的真相是什么？ |
| 6 | 用户视角 | 对普通人意味着什么？ |
| 7 | 趋势预判 | 这件事3-6个月后会有什么影响？ |
| 8 | 激励机制 | 参与方各自的动机是什么？ |
| 9 | 成本分析 | 谁在承担隐性成本？ |
| 10 | 赢家视角 | 这件事对谁最有利？为什么？ |

## 规则

- 每个观点必须有支撑论据（来自调研素材）
- 观点要具体，不要空泛
- 10 个角度必须方向不同，不是同义改写
- 观点要有传播力：要么新信息、要么新视角、要么强判断

## 质量检查

生成后自检：
- [ ] 有没有观点是"正确的废话"？删掉重写
- [ ] 有没有两个角度说的是同一件事？合并或替换
- [ ] 每个观点是否都有具体论据支撑？

## 输出格式

```json
[
  {
    "angle": "反常识",
    "viewpoint": "AI编程工具爆发反而让效率差距变大了",
    "evidence": ["有工具的团队效率提升300%，没工具的还是老样子"]
  }
]
```

## 各角度好案例参考

### 反常识
- 好："不是工具不行，是团队的工作流改不动"
- 差："AI编程工具很重要"（正确的废话）

### 二阶效应
- 好："工具爆发后，第一个被冲击的是技术管理者的价值"
- 差："工具会提升效率"（大家都知道）

### 利益分析
- 好："卖工具的比用工具的更赚"
- 差："对大家都好"
