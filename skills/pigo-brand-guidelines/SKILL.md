# Pigo Brand Guidelines Skill

## 什麼時候用

- 使用者說「幫我寫一個符合 Pigo 風格的貼文」
- 使用者說「這個設計符合品牌嗎」
- 使用者說「品牌的用色規範」
- 使用者說「創作一個品牌相關的圖片」

## 品牌核心元素

### 品牌使命
```
用 AI 技術提升個人與組織的學習、生產力與創造力。
```

### 品牌調性
| 面向 | 描述 |
|------|------|
| 專業性 | 嚴謹、深度、有洞察 |
| 溫度 | 友善、不冷漠、有觀點 |
| 清晰 | 簡潔、不含糊、直接 |
| 前瞻 | 探索新技術、保持好奇 |

### 品牌色彩

| 色彩名稱 | 色值 | 用途 |
|----------|------|------|
| Primary | #3B82F6 | 主要按鈕、連結 |
| Secondary | #8B5CF6 | 強調元素 |
| Background | #FFFFFF | 背景 |
| Surface | #F9FAFB | 卡片背景 |
| Text Primary | #111827 | 主要文字 |
| Text Secondary | #6B7280 | 次要文字 |

### 字體規範

```
主要字體：Inter
備用字體：system-ui, sans-serif

權級：
- H1: 28px, Bold
- H2: 24px, SemiBold
- H3: 20px, SemiBold
- Body: 16px, Regular
- Caption: 14px, Regular
```

## 內容創作規範

### 社群貼文

| 平台 | 長度 | 風格 |
|------|------|------|
| X (Twitter) | 100-280字 | 直接、有觀點、帶標籤 |
| LinkedIn | 150-300字 | 專業、有洞察 |
| 部落格 | 800-2000字 | 深度、有結構 |

### 圖片設計

```
風格要求：
- 簡潔，不要過度裝飾
- 使用品牌色彩
- 保持留白
- 圖標簡單有力
```

## 邊界條件

| 情況 | 處理方式 |
|------|----------|
| 偏離品牌調性 | 提出修改建議 |
| 色彩不合規範 | 提供正確色值 |
| 不確定是否合規 | 列出疑點讓使用者確認 |

## 示例

### 典型使用場景

**輸入：**
> 幫我寫一個推文介紹新的 AI 功能

**輸出：**
```markdown
標題：[新功能名稱]

內容：
[2-3句直接有力的描述，說清楚價值]

#AI #新功能 #[相關標籤]

---
是否符合品牌調性：是
備註：[可選的建議]
```
