# Weather Skill

## 什麼時候用

- 使用者說「今天天氣怎麼樣」
- 使用者說「幫我查一下東京的天氣」
- 使用者說「這週末會下雨嗎」
- 使用者說「台北未來一週的天氣」

## 工作流程

### Step 1: 確認地點

```
必要：
- 地點（預設：台北，或使用者指定）

可選：
- 特定日期（今天、明天、或指定日期）
- 特定時間點
```

### Step 2: 查詢天氣

使用 wttr.in 服務（無需 API key）：

```bash
# 基本查詢（當前天氣）
curl "wttr.in/Taipei?format=3"

# 當前天氣 + 今日預報
curl "wttr.in/Taipei?format=4"

# 未來三天預報
curl "wttr.in/Taipei?format=3&lang=zh"

# 指定日期（明天 = 1, 後天 = 2）
curl "wttr.in/Taipei?format=3&language=zh&1"

# 特定日期（如 2026-04-20）
curl "wttr.in/Taipei?format=3&language=zh&DATE=20240420"
```

### Step 3: 解析與呈現

將 curl 回傳的結果整理成人可讀格式：

```markdown
# [地點] 天氣

## 當前天氣
- 溫度：[溫度]
- 天氣：[晴/陰/雨等]
- 濕度：[濕度%]
- 風速：[風速]

## 今日預報
[圖示] [天氣狀況]
最高：[溫度] / 最低：[溫度]

## 未來數日
| 日期 | 天氣 | 最高 | 最低 |
|------|------|------|------|
| 明天 | ☀️ | 28°C | 22°C |
| 後天 | 🌧️ | 26°C | 21°C |
```

## 輸出格式選項

| 格式 | 參數 | 說明 |
|------|------|------|
| 簡短 | `?format=3` | 一行簡短描述 |
| 詳細 | `?format=4` | 完整預報資訊 |
| UTF-8 | `&lang=zh` | 中文輸出 |

## 支援地點格式

```
# 城市名（自動識別）
Taipei, Tokyo, New York

# 機場代碼
TPE, NRT, JFK

# 地點+國家
Taipei,Taiwan

# 特定座標
London?format=3
```

## 邊界條件

| 情況 | 處理方式 |
|------|----------|
| 地點找不到 | 嘗試更通用的名稱，告知使用者可能的替代方案 |
| 無網路連線 | 告知「目前無法取得天氣資訊」 |
| 地點名稱模糊 | 列出可能的匹配項讓使用者選擇 |
| API 無回應 | 使用備用來源或告知無法取得 |

## 示例

### 典型使用場景

**輸入：**
> 台北今天天氣怎麼樣？

**執行：**
```bash
curl "wttr.in/Taipei?format=3&lang=zh"
```

**輸出：**
```
台北：25°C，多雲
```

格式化後：
```markdown
# 台北天氣

## 當前
- 溫度：25°C
- 天氣：多雲
- 濕度：78%
- 風速：12 km/h

## 今日預報
☁️ 多雲
最高：28°C / 最低：22°C
```

## 限制說明

- 依賴 wttr.in 服務，準確率依賴該服務
- 國外城市可能沒有完整資料
- 預報通常只提供 3-7 天
- 即時天氣可能與實際有輕微偏差
