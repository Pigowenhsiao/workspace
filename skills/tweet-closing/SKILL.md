# 结尾转化器

为推文生成 5 种结尾，每种对应不同的目标动作：

## 5 种类型

| # | 类型 | 目标 | 示例 |
|---|------|------|------|
| 1 | 提问型 | 回复 | "你怎么看？" "你遇到过类似的情况吗？" |
| 2 | 开放型 | 讨论 | 留一个没回答的问题，给出不完整的判断 |
| 3 | 转发型 | 转发 | "值得你身边的人也看看" 说一个别人也会遇到的问题 |
| 4 | 关注型 | 关注 | "这个话题我会持续跟踪" 暗示后续还有更深入的分析 |
| 5 | 行动型 | 私信/点击 | "如果你想XX，可以试试" 给出一个具体的下一步 |

## 规则

- 结尾**不超过 2 句**
- 不硬推，像自然收束
- 和正文风格一致（老程序员风格就用老程序员口吻结尾）
- **不加**"关注我获取更多"这种话
- **不加**emoji堆砌
- 每种类型给 **2-3 个备选**

## 输出格式

```json
[
  {
    "type": "提问型",
    "goal": "reply",
    "options": ["你身边有这种例子吗？", "你用AI编程工具多久了？效果怎么样？"]
  },
  {
    "type": "开放型",
    "goal": "discussion",
    "options": ["但问题是，工作流改不动是技术问题还是管理问题？", "工具爆发之后，接下来会发生什么？"]
  },
  {
    "type": "转发型",
    "goal": "retweet",
    "options": ["转给做AI产品的朋友，这个数据有用", "你团队也在用老方法？转给需要的人"]
  },
  {
    "type": "关注型",
    "goal": "follow",
    "options": ["AI编程工具这个话题我会持续跟踪", "后续会发更多关于工具采纳率的实际案例"]
  },
  {
    "type": "行动型",
    "goal": "click",
    "options": ["想提升团队采纳率？先从一个人开始，给他2周时间", "我整理了一个工具清单，想要的私信"]
  }
]
```
