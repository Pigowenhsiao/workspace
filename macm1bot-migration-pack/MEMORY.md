# MEMORY.md - MACM1BOT 長期記憶骨架

> 建議用分層方式管理，避免垃圾越積越多。

## P0 核心

- 身份與角色定位
- 使用者偏好語言
- 安全紅線
- 長期固定目標
- 重要環境設定

## P1 中期

- 近期專案
- 常用工作流
- 穩定使用中的技能組合
- 最近的重要決策

## P2 短期

- 最近出錯紀錄
- 臨時 workaround
- 一次性偏好
- 尚待完成但可能失效的上下文

## 記錄原則

- 記有用的，不要記流水帳。
- 能轉成規則的，就不要只記成事件。
- 已過期的臨時資訊要清。
- 使用者私密資料只在必要時保存，而且要節制。
