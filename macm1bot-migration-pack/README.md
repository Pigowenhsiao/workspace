# MACM1BOT 移植包

這份移植包的目標，不是把 Jarvis 的「靈魂魔法」硬拷貝，而是把真正可移植的部分整理出來：

- 做事流程
- 回覆風格
- 工作邏輯
- skill 組合建議
- 安裝與驗證清單

## 內容

- `AGENTS.md`：MACM1BOT 的工作規則
- `SOUL.md`：MACM1BOT 的人格與回覆風格
- `IDENTITY.md`：角色定位
- `TOOLS.md`：本地環境備忘模板
- `MEMORY.md`：建議的長期記憶骨架
- `skills-plan.md`：建議 skill 組合與用途
- `install-checklist.md`：安裝與驗證步驟

## 使用方式

1. 先找到 MACM1BOT 的 workspace。
2. 備份原本檔案。
3. 把這份移植包內對應檔案複製過去。
4. 依 `skills-plan.md` 安裝缺少的 skills。
5. 依 `install-checklist.md` 驗證是否生效。

## 重點

這份包移植的是：
- 可重複使用的規則
- 可維護的文件
- 可安裝的 skill 組合

這份包**不保證**移植的是：
- 純經驗型判斷
- 所有上下文記憶
- 每次都完全一致的語感

但老實講，真正有價值的 80%，已經都在這裡了。
